Overview

This program is a terminal-style interface with the following features:

  • Launch websites (custom & guest Chrome sessions).
  • Anime Vault ‚Äî encrypted list you can edit and copy from.
  • Clicky Clicker ‚Äî a macro tool for keys, media controls, and shortcuts.
  • Password protection with lockout on repeated failed attempts.

Where files live (important)

  • %AppData%\MainGUI ‚Äî main folder the program creates and uses.
  • logons.dll ‚Äî Anime Vault (encrypted).
  • logoffs.dll ‚Äî encrypted password file.
  • configs.cfg ‚Äî app settings (lock timer, max fails, profile number, etc.).
  • click_config.cfg ‚Äî key mapping for Clicky Clicker.
  • sites.cfg and authorize.cfg ‚Äî custom and guest site lists (plain text, one URL per line).
  • systems32.sys ‚Äî fail count file.
  • winlogxs.dll ‚Äî lock file.
  • knernals32.txt ‚Äî log file for wrong password attempts.
  • oooh.bat ‚Äî helper batch that logs wrong password attempts safely.

Getting Started

First run

  1. Double-click the program EXE to run.
  2. The first run will create %AppData%\MainGUI and default files if missing.
  3. If asked for a password and it is unchanged, the default is 1234. You are recommended to change it.

Open / Exit / Restart

  • The program displays a main GUI labeled ENCRYPTED TERMINAL. Click a button to enter a feature.
  • Exit the program via the Exit Terminal button.
  • To reload after edits, run the EXE again.

Main Menu ‚Äî Buttons explained

  • Access (Retrieve / Submit) ‚Äî opens Anime Vault options to view, edit, or copy titles.
  • Launch Mission Interface ‚Äî Chrome launcher for opening websites (custom profile or guest).
  • Start Clicky Clicker ‚Äî enables the Clicky Clicker macro tool.
  • Settings ‚Äî opens the Settings panel for lockdown timer, profile number, and site list editors.
  • Exit Terminal ‚Äî closes the program.

Password & Security

The program protects access with an encrypted password and maintains a fail counter and temporary lock file.

  • Password is stored encrypted in logoffs.dll.
  • systems32.sys tracks consecutive bad attempts.
  • Exceeding max fails writes a timestamp to winlogxs.dll for temporary lock.
  • Default settings: Lockdown timer = 60s, Max failed attempts = 3, Profile number = 7, animations disabled by default.

Change password

  1. Open Settings ‚Üí Change Access Key.
  2. Enter the old password and the new password twice to confirm.
  3. The new password is saved encrypted.

If you lock yourself out

Delete %AppData%\MainGUI\winlogxs.dll to remove the lock. Password remains unchanged.

How encryption works (simple)

Titles and password use a simple letter-substitution cipher. You don't need to understand this to use the program.

  1. Open Access ‚Üí Edit Anime List.
  2. Enter one title per line.

Chromes / Launchers

Two launch types:

  • Custom Sites ‚Äî opens URLs in the Chrome profile defined by Profile Number.
  • Guest Sites ‚Äî opens URLs in a guest Chrome session.

Usage

  1. Open Chromes from the main menu.
  2. Select sites and click GOOOO (custom) or SUIIII (guest).

Tips

  • Add sites manually via Settings ‚Üí Manage Custom/Guest Sites.

Clicky Clicker

A macro helper that watches keys for actions:

  • Volume Up / Down
  • Swap / Alt+Tab mix

Configuration

Keys are mapped in click_config.cfg. One key per line.

FAQ

Q: Can I edit config files manually?

A: Yes for sites.cfg and click_config.cfg. Use the program for password or Anime Vault.

Q: Lost password?

A: Replace logoffs.dll with a new encrypted password or delete it to reset to default 1234 (advanced).

Q: Why strange filenames like .dll or .sys?

A: They are just storage files. They are not real system files and do not affect Windows system components.
